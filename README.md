# S-DES
这是一个S-DES算法实现
## **开发说明**
根据"信息安全导论"课程第5次课讲述的DES算法：

* 实现简单的加解密（二进制加解密和ASCII码字符串加解密）
* 对相同的相同的密文进行交叉测试
* 在已知明密文的条件下暴力破解出密钥
* 对随机的明密文对进行封闭测试

## **运行程序**
```bash
sd_des.html
```
